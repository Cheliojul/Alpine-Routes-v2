# Alpine Routes — v2

Premium mountain transport & transfer company landing page.

A single, self-contained `index.html` (HTML + CSS + JS, no frameworks, no build step).
Open the file directly in a browser, or view the live version on GitHub Pages.

## Features
- Cinematic alpine-night hero with mouse parallax, staggered entrance, and atmospheric snow
- Signature elevation-profile route cards (real altitude climb per transfer)
- Frosted-glass "Why" value props
- Sticky frosted navigation with mobile menu
- Fully responsive and `prefers-reduced-motion` aware

## Develop
Just edit `index.html` and refresh the browser.

## Deploy
Hosted via GitHub Pages from the `main` branch (root).
